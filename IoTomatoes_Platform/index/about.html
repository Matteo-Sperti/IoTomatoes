IoT Platform

Introduction

The proposed IoT platform for Smart Farming is built on the base of the microservices design pattern. It will basically exploit 2 types of communication paradigms:

  • PUBLISH/SUBSCRIBE: based on the usage of the MQTT protocol;
  • REQUEST/RESPONSE: based on REST Web Services.

Components

Launcher

The docker-compose.yml file is a script that allows to build and run all the microservices of the IoT platform in a single command. It is based on the Docker Compose tool, which is a tool for defining and running multi-container Docker applications. The docker-compose.yml file is a YAML file that defines the services that make up the application as well as the configuration needed to run each service.

Microservices

The IoT platform is composed of 4 microservices:

  • IoT Gateway: it is the main component of the IoT platform. It is responsible for the communication with the IoT devices and the communication with the other microservices of the platform. It is based on the Mosquitto MQTT broker.
  • IoT Data Manager: it is responsible for the management of the data collected by the IoT devices. It is based on the Mosquitto MQTT broker.
  • IoT Data Analyzer: it is responsible for the analysis of the data collected by the IoT devices. It is based on the Mosquitto MQTT broker.
  • IoT Data Visualizer: it is responsible for the visualization of the data collected by the IoT devices. It is based on the Mosquitto MQTT broker.

IoT Devices

The IoT platform is able to communicate with different types of IoT devices. In particular, it is able to communicate with:

  • Arduino: it is a microcontroller board based on the ATmega328P microcontroller. It is able to communicate with the IoT platform through the MQTT protocol.
  • ESP8266: it is a low-cost Wi-Fi microchip with full TCP/IP stack and microcontroller capability produced by Espressif Systems. It is able to communicate with the IoT platform through the MQTT protocol.
  • ESP32: it is a series of low-cost, low-power system on a chip microcontrollers with integrated Wi-Fi and dual-mode Bluetooth. It is able to communicate with the IoT platform through the MQTT protocol.
  • Raspberry Pi: it is a series of small single-board computers developed in the United Kingdom by the Raspberry Pi Foundation to promote the teaching of basic computer science in schools and in developing countries. It is able to communicate with the IoT platform through the MQTT protocol.
